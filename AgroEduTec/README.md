# AgroTec - TecnoAgro Educativo

Prototipo funcional desarrollado en Wix con Velo.

## Descripción
Plataforma educativa para agricultores enfocada en tecnologías agrícolas modernas.

## Tecnologías usadas
- Wix Studio
- Velo by Wix (JavaScript)
- HTML embebido
- Google Maps integration

## Funcionalidades
- Navegación por secciones
- Botones interactivos
- Formularios de contacto
- Sección de servicios
- Blog educativo
- Formación digital
